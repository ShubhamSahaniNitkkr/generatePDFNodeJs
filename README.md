Generate PDF invoices from HTML form input using Node and a React UI.

## Use Cases

- PDF generation pipeline
- Form-to-document workflow
- Full-stack file download

## Tech Stack

- Node.js
- Express
- html-pdf
- puppeteer
- React

## How to Run Locally

```bash
npm install && npm run uiinstall && npm run demo
```
